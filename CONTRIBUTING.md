# Contributing to ApexSpiral

## Code Style
- Follow APEX code standard
- Add APEX comment blocks to functions

## Testing
- Python: pytest tests/test_apex_v10.py
- Rust: cargo test

## Commit Messages
- feat: new feature
- fix: bug fix
- docs: documentation
- test: tests
- chore: maintenance

## Pull Request Process
1. Fork the repository
2. Create a feature branch
3. Run tests locally
4. Submit PR
